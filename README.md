# vizitka-site

Сайт-визитка (статический, без бэкенда и базы данных).

## Инфраструктура

- Репозиторий: этот, `oleksiikovalskyi/vizitka-site` на GitHub, ветка `main`.
- Хостинг превью: [Vercel](https://vercel.com) — подключен к этому репозиторию,
  автоматически пересобирает и деплоит сайт при каждом пуше в `main`.
- Продакшн-ссылка: **https://vizitka-site-eight.vercel.app**

## Как это работает

1. Любые изменения (страницы, стили, скрипты) коммитятся и пушатся в `main`.
2. Vercel сам подхватывает пуш и обновляет сайт по ссылке выше — обычно за
   несколько секунд, ничего вручную запускать не нужно.
3. Для превью конкретной ветки/PR (если понадобится) Vercel создаёт отдельную
   preview-ссылку — см. вкладку Deployments в Vercel dashboard.

## Для будущей сессии (Claude Design / Cowork / Code)

- Клонировать репозиторий можно анонимно (публичный, только для чтения):
  `git clone https://github.com/oleksiikovalskyi/vizitka-site.git`
- Для пуша нужен push-доступ — токен лежит на компьютере пользователя вне этого
  репозитория (не коммитить токены в код).
- `index.html` сейчас — заглушка «Сайт в разработке», ждёт реальный дизайн/контент.
