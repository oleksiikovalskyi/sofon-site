# sofon-site

Сайт компанії **Sofon** (ТОВ «СОФОН») — інженерно-виробничий завод обладнання для
ліній розливу напоїв і води. Статичний сайт: голий HTML/CSS, без CMS і бекенду.

## Статус

Фаза 7 (збірка), старт. Готові сторінки (UA):
- `/` — головна (нове позиціонування: флагман — **Оснастка**)
- `/products/osnastka/` — оснастка (форматні деталі), флагман
- `/contract-manufacturing/` — контрактне виробництво (**чернетка** — факти уточнюються)

Далі: решта продуктових сторінок, About / News / Contact, дзеркало EN під `/en/`.

## Структура

- `index.html`, `products/<line>/index.html`, `contract-manufacturing/index.html` — сторінки
- `css/styles.css` — дизайн-система (IBM Plex Sans/Mono, Amber accent `#F07C1A`, сітка 1120/12)
- Шапка й футер продубльовані в кожній сторінці (статика, без інклюдів)

## Інфраструктура

- Репозиторій: `oleksiikovalskyi/sofon-site`, гілка `main`.
- Хостинг: [Vercel](https://vercel.com) — авто-деплой при кожному пуші в `main`.
- Продакшн-лінк: **https://vizitka-site-eight.vercel.app**
  (автодомен не змінився після перейменування репо — косметика, можна дати кастомний аліас).

## Як оновлювати

1. Правки коммитяться і пушаться в `main`.
2. Vercel сам пересобирає за кілька секунд.
3. Токен для пуша — на комп'ютері користувача, поза репозиторієм (не коммітити токени).
